# 🚚 **아웃소싱 배달앱 프로젝트**
- `Spring Boot`를 활용한 '배달앱' 협업 프로젝트 입니다.
- 개발기간: 25.01.07 ~ 25.01.13


✨ **주요 기능**
- 고객(CUSTOMER) / 사장님(OWNER) 으로 **회원가입**, **로그인**
- 사장님 권한으로 **가게**/**메뉴** 관리 기능
- 고객이 메뉴 **주문**을 하면 사장님이 수락을 하고 `조리중 → 배달중 → 배달 완료` 순으로 배달 상태 변경
- 배달 완료된 주문건에 대해서 **별점**(1~5점)으로 리뷰 가능
